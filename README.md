## Project Overview
This project focuses on building a **professional and interactive sales dashboard**
using **Seaborn** for statistical visualizations and **Plotly** for interactive charts.
The dashboard provides insights into **sales trends, product performance, pricing
distribution, and regional sales behavior**.

---

## Objectives
- Analyze sales data using visual analytics
- Understand data distribution and correlations
- Build interactive and static visualizations
- Design a clean, cohesive dashboard layout
- Present insights in an industry-standard format
